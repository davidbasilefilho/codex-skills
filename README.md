# codex-skills

A curated collection of reusable agent skills for planning, implementation, debugging, testing, architecture, documentation lookup, teaching, and handoffs.

## Included skills

- `controlled-parallel-work`
- `execute-approved-plan`
- `plan-and-stop`
- `context7`
- `improve-codebase-architecture`
- `tdd`
- `handoff`
- `diagnose`
- `teach`

## Install

<details>
<summary>Bun (recommended)</summary>

```bash
bunx skills add davidbasilefilho/codex-skills
```

</details>

<details>
<summary>npm</summary>

```bash
npx skills add davidbasilefilho/codex-skills
```

</details>
